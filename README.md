# 특별실 예약

초등학교 특별실을 교시 단위로 예약하는 간편 웹앱입니다.

## 기능

- 주간 시간표 형태로 예약 조회
- 학년, 반, 특별실, 날짜, 교시로 예약
- 같은 날짜, 같은 교시, 같은 특별실 중복 예약 방지
- 예약할 때 입력한 삭제 비밀번호로 예약 삭제
- 특별실 필터
- 이전 주, 이번 주, 다음 주 이동

## 기본 특별실

- 창의놀이실
- 청계누리(강당)
- 컴퓨터실
- 음악실
- 다모임실

## 설치

노드 22 이상을 권장합니다.

```bash
npm install
```

## 개발 실행

```bash
npm run dev
```

화면은 보통 `http://localhost:5173`에서 열립니다. 서버는 `http://localhost:3001`에서 실행됩니다.

## 실제 실행

```bash
npm start
```

저장 기능까지 쓰려면 이 명령으로 노드 서버가 실행 중인 주소에서 열어야 합니다. 깃허브 저장소 화면이나 정적 파일 서버에서는 예약을 저장할 수 없습니다.

## 확인

```bash
npm test
npm run build
```

## 자료 저장

예약 자료는 `data/reservations.json`에 저장됩니다. 삭제 비밀번호는 그대로 저장하지 않고 확인값으로 저장되며, 예약 목록 응답에도 포함되지 않습니다.

깃허브 저장소 화면이나 정적 배포만으로는 예약 저장이 되지 않습니다. 저장 기능은 노드 서버의 `/api/reservations`가 실행 중일 때만 동작합니다.
