# 특별실 예약

초등학교 특별실을 교시 단위로 예약하는 간편 웹앱입니다.

## 기능

- 주간 시간표 형태로 예약 조회
- 학년, 반, 특별실, 날짜, 교시로 예약
- 같은 날짜, 같은 교시, 같은 특별실 중복 예약 방지
- 예약할 때 입력한 삭제 비밀번호로 예약 삭제
- 특별실 필터
- 이전 주, 이번 주, 다음 주 이동

## 기본 특별실

- 창의놀이실
- 청계누리(강당)
- 컴퓨터실
- 음악실
- 다모임실

## 설치

노드 22 이상을 권장합니다.

```bash
npm install
```

## 개발 실행

구글 시트 저장소를 연결하려면 `.env`에 앱스 스크립트 웹앱 주소를 설정합니다.

```bash
cp .env.example .env
```

`.env` 파일의 `VITE_GOOGLE_SCRIPT_URL` 값을 실제 앱스 스크립트 웹앱 주소로 바꿉니다.

```bash
npm run dev
```

화면은 보통 `http://localhost:5173`에서 열립니다.

## 구글 시트 저장소

예약 자료는 구글 시트에 저장됩니다. 설정 방법은 [google-apps-script/README.md](google-apps-script/README.md)를 참고하세요.

## 확인

```bash
npm test
npm run build
```

## 자료 저장

예약 자료는 연결한 구글 시트에 저장됩니다. 삭제 비밀번호는 그대로 저장하지 않고 확인값으로 저장되며, 예약 목록 응답에도 포함되지 않습니다.

## 기존 로컬 서버 코드

이전 파일 저장 서버 코드도 저장소에는 남아 있습니다. 다만 현재 화면 앱의 기본 저장소는 구글 시트입니다. 정식 운영은 구글 시트 방식을 사용하세요.

```bash
npm start
```
